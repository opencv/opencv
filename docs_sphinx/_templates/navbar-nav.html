{# Override of pydata_sphinx_theme's components/navbar-nav.html. PyData's default renders both top-level toctree items AND the configured external_links in one bar. For OpenCV we want ONLY the Doxygen-style external nav (Main Page, Related Pages, Namespaces, …) in the header — the tutorial module tree belongs in the left sidebar, not the header. So we render external_links only. conf.py builds each link's URL from DOXYGEN_BASE_URL (default https://docs.opencv.org/5.x/) + a filename. By default we rewrite those to a depth-aware relative path into the local Doxygen output at /doc/doxygen/html/. Exception: "Main Page" routes to the Sphinx master_doc (tutorials/tutorials) via `pathto`, because the Sphinx-rendered site IS the redesigned landing page — sending the user to the legacy Doxygen index would feel like a regression. The `+ 2` in _doxygen_local covers climbing out of html/ and docs_sphinx/ to reach build/doc/doxygen/html/. #} {%- set _depth = (pagename or '').count('/') %} {%- set _doxygen_local = ('../' * (_depth + 2)) ~ 'doc/doxygen/html/' %}
